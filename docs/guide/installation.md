## 安装组件

推荐以 yarn 为包管理工具，速度更快。

::: code-group

```sh [yarn]
$ yarn add gov-ui
```

```sh [npm]
$ npm install gov-ui
```

```sh [pnpm]
$ pnpm add gov-ui
```

```sh [bun]
$ bun add gov-ui
```
:::
